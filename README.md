# Интернет-магазин "MusicShop"
Учебный проект для колледжа, не реальный магазин.

### Как запустить?
- Скачайте и распакуйте .zip архив
- Установите Python 3.10+
- Установите зависимости из requirements.txt (Открыть папку проекта в терминале и ввести команду "pip install -r requirements.txt")
- Запустите app.py (Открыть папку проекта в терминале и ввести команду "python app.py")
